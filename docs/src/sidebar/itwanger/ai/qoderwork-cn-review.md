## 01、安装 QoderWork CN

访问 `https://qoder.com.cn/qoderwork`

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623202805.png)

macOS 和 Windows 两个版本都支持哈。注册登录后的界面如下所示，非常清爽哈～

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623200810.png)

左侧菜单包括：

①、专家套件，面向特定行业，特定人群，意味着你可以通过 QoderWork 做产品设计、合同管理、股权投资这些。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623203646.png)

②、技能，可以理解为 QoderWork 的 SkillHub，里面有很多第三方开发的技能，涵盖文档处理、数据分析、图像处理等。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623203932.png)

③、连接器，可以连浏览器、QoderWork 国际版、控制你的电脑等

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623204104.png)

④、定时任务，一些需要固定周期处理的事情，比如说每天下班时帮你自动整理文件夹，写周报等等。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623204222.png)

⑤、IM 频道，可以通过钉钉、飞书、微信等 IM 工具直接和 QoderWork 互动，发起任务、查看结果。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623204528.png)

提醒一点，工作文件夹建议选一个独立目录，做好区分。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623204717.png)

普通文件任务建议限定在授权工作目录内；开启浏览器、计算机控制或连接器后，以对应系统权限和连接器授权范围为准。

## 02、工作台

QoderWork 针对高频场景做了四个工作台（通用、设计、写作、幻灯片），输入框左侧底部可以自由切换。


![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260624091040.png)


设计工作台的思路是 Code as Design。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623231748.png)

内置了 100 多种风格参考，这样你的产品在诞生之初就很成熟了。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623231655.png)

选择【设计】工作台的时候，我们可以选择不同的风格作为参考，模板都漂亮得很。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623232347.png)

给大家看一眼，这个设计真的很漂亮，并且画布还可以针对某个元素进行修改，如果你觉得不满意的话（嗯，我觉得很满意不用修改😄）。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623233624.png)

## 04、技能市场

技能（Skill）是 QoderWork 的扩展机制，相当于一份预先编排好的工作指南。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623222007.png)

一个技能的结构很简单，就是一个文件夹，里面放一个 `SKILL.md` 文件，用自然语言写清楚做什么、什么时候触发、怎么执行。存储路径在 `~/.qoderworkcn/skills/` 目录下。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623222125.png)

QoderWork 的 Skill 格式兼容 Claude Code Skills，意味着之前在 Claude Code 写过的 Skill 可以直接迁移过来用，零成本切换。

我们就拿之前生成好的PRD结合 UI 设计来演示一下。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623222442.png)

提示词非常简单，就是把之前的PRD文件放到新任务的上下文窗口。

我们来看看产物，覆盖了 PRD 中 MVP 的 8 个核心页面。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623224212.png)

这里我们给 JSX 文件做了 HTML 转换，方便给大家展示。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623224156.png)

整体设计系统以暖琥珀色为主色调传递信任感，翡翠绿标注认证/完成状态，玫红色突出价格和紧急操作。

## 06、定时任务

支持六种触发频率，包括单次执行、固定间隔、每小时、每天、每周、每月。

创建方式很直观，用自然语言描述就行。

比如输入"每天下午 6 点帮我整理下载文件夹，按文件类型分类归档"，QoderWork 自动识别频率和任务内容，生成定时计划。也可以在定时任务面板里手动配置参数。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623230732.png)

每次触发会创建一条独立对话记录。触发时可以调用已安装的 Skill、使用连接器操作外部工具、读写本地文件，能力和手动发起任务一致。

一个限制需要注意，设备必须保持唤醒状态。QoderWork 提供了"保持系统唤醒"选项，但这意味着电脑得一直开着。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623230842.png)

定时任务可以和 IM 频道配合使用，比如设定"每天早上 9 点生成数据日报，发送到钉钉群"，全自动执行，人不用在电脑前。

## 08、应用快照、语音输入和意识

应用快照是一个很实用的小功能。

双击 Command 键，QoderWork 自动截取当前前台窗口的画面发送给 AI 作为上下文。看到一个网页想让 AI 分析，双击 Command 就行，不用手动截图再粘贴。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623231845.png)

比如说我在浏览器里正在学习Claude Code 的上下文机制和 Memory 机制，我按下键盘两侧的 Command 键。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623232226.png)

QoderWork 就会把当前网页截图发给 AI，AI 会自动分析网页内容，提取关键信息

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623232122.png)

语音输入方面，设计工作台支持直接语音描述需求，对话区支持上传音频文件自动转录。

"意识"功能整合了记忆、反思、技能进化三个模块。

![](https://cdn.paicoding.com/stutymore/qoderwork-cn-review-20260623215841.png)

日常对话积累短期记忆，跨会话有价值的内容晋升长期记忆，所有记忆以 Markdown 存放在本地，可以查看和编辑。

检测到重复任务时会自动建议生成 Skill，用户可以采纳、忽略或驳回，驳回后自动冷却不再打扰。

技术上使用 Fork Session 实现，进化任务从主会话分叉出轻量子会话，复用原始上下文，额外成本控制在主对话总开销的 5% 以内。
